Steps to Follow


Step 1: Select a single indicator (HSTP II or Global PHC Indicator) from the Main Page or from the right side navigation menu


Step 2: After Selecting any indicator in the above step (1) Click on Tree Header Drop Down as per the selected Indicator and select a single Indicator Category from Equity, Finance, Performance or All in one selector (Displays all of the Selected main Indicator Dashboard in one page)


Step 3: After the Selecting a single Indicator A Dashboard will auto load and it will display various inline filters such as

- Filter by Region

- Year Filter

- Filter by Indicator Name

- Source Filter

- Hovering

- Time Range (Series) Filter (Beta/Testing/Experimental)


Step 4: Using Filters to Narrow down the information displayed: Use the inline the available filters by clicking on their corresponding filter headers and you can add multiple or single filters on regions, indicators and click on Add Filter when you are done selecting each of your filters then the dashboard will automatically display the dashboard visualization as per your selection. Additionally you can use the time series that are available at the top of the dashboard by hovering your mouse/pointer and it will display the data available based on your time series hover if thereâ€™s any data available.


Step 5: Printing, exporting and saving your visualized data done on the above Step (4) on the upper right corner you will see the print/save button and click on it and a window will show you to print on hard copy or save it as a local PDF file and follow on the screen prompt to proceed to your desired output and after you have finished printing/saving the screen will redirect you to home of your previously selected Indicator home page and if you want to continue to use the dashboard please use the steps mentioned in step (2) through step (4).


Step 6: If you want to go home or change indicators from your current dashboard or page please navigate through the right side Navigation Menu.


Step 7: If you would like to share or embed the current or selected dashboard to anyone or for your future reference please click on share button and the system will automatically provide a sharing or embedding link to your clipboard and you can go ahead and paste the clipboard to your desired place.


Step 8: (Beta/Testing/Experimental) For Exporting data for the selected dataset/indicator please go to the all in one dashboard and filter your desired dataset from the data filters available from the in line filters displayed at the header and then click on Export.


Step 9: If you are having trouble navigating through the user interface while on mobile device please make it Desktop Mode and if you are on a desktop/pc/laptop but still having trouble viewing the User Interface please use the lines on the right side or at the bottom to resize the display output.


Step 10: If you would like to filter or over view using Time Series / Range Please Hover on the Time Line Indicators listed with color identifiers at the top of the displayed dashboard or use the time range inline filter to filter using your desired time series. (*)


Step 11: If you would like to see data set per graph or inside the dashboard you can hover on the visualized data set and the system will show you the specific data values for the visualized dashboard.
